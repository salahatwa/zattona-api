## Features

-[x] Log page
-[x] Article directory and configurable
-[x] Friends link page
-[x] Page title can be configured
-[x] Adapt to mobile
-[x] Search function
-[x] Code line number display
-[x] Linked pages are displayed in groups
-[x] Album page
-[x] Adapt to halo 1.3.x

-[x] Global PJAX
-[x] Night mode
